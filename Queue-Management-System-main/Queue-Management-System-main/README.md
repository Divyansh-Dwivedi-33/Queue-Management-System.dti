# BookMyTable 🍽️

BookMyTable is a Django-based web application that allows users to explore menu items and book tables at a restaurant. The system provides an easy-to-use interface for customers and an admin panel for managing bookings and menu data.

---

## 🚀 Features

* User Registration & Login
* Table Booking System
* View & Manage Reservations
* Menu Display with Categories
* Admin Panel for Management

---

## 🛠️ Tech Stack

* Python (Django Framework)
* HTML, CSS, Bootstrap
* SQLite Database
* Django Templates

---

## ⚙️ Installation & Setup

1. Clone the repository

```
git clone https://github.com/your-username/BookMyTable.git
```

2. Navigate to project folder

```
cd BookMyTable
```

3. Create virtual environment

```
python3.11 -m venv venv
source venv/bin/activate
```

4. Install dependencies

```
pip install -r requirements.txt
```

5. Run migrations

```
python manage.py migrate
```

6. Start server

```
python manage.py runserver
```

---

## 🌐 Run the Project

Open in browser:
http://127.0.0.1:8000
